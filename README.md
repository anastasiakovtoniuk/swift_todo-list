# swift_todo-list

### команди для запуску

> для першого запуску також треба спочатку забілдити проект (swift build) після переходу в директорію проекту

* swift run todo list  = виведе увесь список завдань
* swift run todo add "назва таски"  = додавання нової таски (також локально зберігається у туду.джсон)
* swift run todo done 1   = зміна статусу завдання по номеру завдання
* swift run todo list --pending   = покаже тільки завдання зі статусом пендінг
* swift run todo clear --yes   = очистити увесь список завданнь (обовʼязково треба ввести --yes яке підтвердужє очищення, якщо не ввести - буде validation error)
* swift run todo remove 2 = видалення окремої таски за номером
